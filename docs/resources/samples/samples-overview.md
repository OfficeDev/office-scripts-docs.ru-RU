# <a name="office-scripts-samples-and-scenarios"></a>Примеры и сценарии сценариев Office

В этом разделе [содержатся решения автоматизации](../../overview/excel.md) на основе сценариев Office, которые помогают конечным пользователям автоматизировать повседневные задачи. Он содержит реалистичные сценарии, с которыми сталкиваются бизнес-пользователи, и предоставляет подробные решения, а также пошаговые инструкциональные видеосвязи.

Для каждого из проектов в разделе [](#basics) "Основные [](#beyond-the-basics)и не только" ознакомьтесь с исходным кодом, пошаговые видео [**на YouTube**](https://www.youtube.com/playlist?list=PLr3zVPZrMOUMl88fs8uc2GGAePRnNe6m0) и т. д.

В [сценариях](#scenarios) мы добавили несколько более крупных примеров сценариев, демонстрирующих реальные варианты использования.

Мы также будем рады [вкладам от сообщества](#community-contributions-and-fun-samples). Эти примеры являются открытый код.

> [!IMPORTANT]
> Перед началом работы с примерами убедитесь, что выполнены предварительные требования для сценариев Office. Требования к подписке и учетной записи Microsoft 365 см. в разделе "Требования" статьи "Сценарии [Office для Excel"](../../overview/excel.md#requirements).

## <a name="scenarios"></a>Сценарии

Сценарии Office могут автоматизировать части ежедневной процедуры. Эти повседневные задачи часто существуют в уникальных экосистемах с книгами Excel, которые настраиваются определенными способами. Эти более крупные примеры сценариев демонстрируют такие реальные варианты использования. Они включают как сценарии Office, так и книги, чтобы вы могли видеть сценарий от начала до конца.

| Сценарий | Details |
|---------|---------|
| [Анализ загруженного из Интернета](../scenarios/analyze-web-downloads.md) | Этот сценарий содержит сценарий, который анализирует записи веб-трафика для определения страны происхождения пользователя. В нем демонстрируются навыки анализа текста, использования подфункции в скриптах, применения условного форматирования и работы с таблицами. |
| [Извлечение и построение графика данных об уровне воды от NOAA](../scenarios/noaa-data-fetch.md) | В этом сценарии сценарий Office используется для извлечения данных из внешнего источника (базы данных [NOAA Tides и Currents](https://tidesandcurrents.noaa.gov/)) и графирования полученных данных. В нем выделены навыки использования для `fetch` получения данных и использования диаграмм. |
| [Калькулятор оценок](../scenarios/grade-calculator.md) | Этот сценарий содержит скрипт, который проверяет запись преподавателя на соответствие его классам. В нем демонстрируются навыки проверки ошибок, форматирования ячеек и регулярных выражений. |
| [Планирование собеседований в Teams](../scenarios/schedule-interviews-in-teams.md) | В этом сценарии показано, как использовать электронную таблицу Excel для управления временем собеседования и создания потока для планирования собраний в Teams. |
| [Напоминания о задачах](../scenarios/task-reminders.md) | В этом сценарии сценарий Office используется в потоке Power Automate для отправки напоминаний коллегам об обновлении состояния проекта. В нем выделены навыки интеграции Power Automate и передачи данных в скрипты и из них. |
